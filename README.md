# Sales_profit_dashboard

Sales & Profit Dashboard (Excel)
Problem Statement

The company has raw sales data but lacks a clear view of performance across regions, products, customers, and states. A dashboard is needed to quickly analyze sales, profit, orders, and product performance.

Overview

This project converts raw sales data into an interactive Excel dashboard using Pivot Tables, Pivot Charts, and slicers. The dashboard provides a quick understanding of business KPIs, region performance, product profitability, and shipment distribution.

Dataset Summary

Total records: 1952 orders

Includes: Region, State, City, Segment, Category, Product Name, Sales, Quantity, Profit, Discount, Ship Mode, Dates

Purpose: Analyze sales & profit trends across multiple dimensions

Dashboard Highlights

KPIs: Total Sales, Total Profit, Total Orders, Return Rate

Region-wise Sales

Shipment Mode Distribution

Top 3 Least Profitable Products

Interactive Slicers: Segment, Category, Ship Mode

These visuals help quickly identify high-performing regions, profitable products, and customer/shipping behavior.
